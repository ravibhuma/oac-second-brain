## Key Concepts

### AI Assistants — Three Modes
1. Homepage AI Assistant — start with NL question, generates workbooks
2. Workbook AI Assistant — iterative visual design
3. Mobile AI Assistant — voice + podcast-style narration

### AI Agents
A step beyond assistants:
- Combine enterprise data + organizational knowledge via RAG
- Custom knowledge sources (PDFs, reports, policies)
- Explain not just metrics but the "why"

### MCP Server Integration
Standardized operations: discover, retrieve metadata, execute governed queries.
"Open AI integration without compromising security."

### Architecture Principle
Unified platform — same governed foundation across all AI surfaces:
- Semantic Model = trusted business abstraction
- RLS, role-based access enforced everywhere

### Strategic Quote
> "Analytics is no longer something users navigate. It's something they talk to."

### Capabilities Across the Ecosystem
- Context-aware dialogue (follow-up question handling)
- Smart semantic search (catalog + public knowledge)
- Role-based access control + RLS
- User feedback mechanism for AI improvement
